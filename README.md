pcSummerNoteFix
============

Fixes summernote problem while saving content with smarty tags.
Free module for OXID eshop and 6.x.


Installation
1. ```composer require proudcommerce/pcsummernotefix``` 
2. activate module pcSummerNoteFix in shop admin
